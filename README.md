# Simple GWA-Calculator
This project is a Python-based Grade Weighted Average (GWA) calculator that allows students to calculate, save, and view their quarterly grades. Users can input the number of subjects and their corresponding grades, adnd the program computes the average for that quarter. This tool is easy to use, light, and can help students monitor their progress.

# Features:
 - Calculate GWA for any number of subjects
 - Assign letter grades based on numeric GWA
 - Save GWA for 1st, 2nd, 3rd, and 4th quarters
 - View saved records in a simple menu interface

 1. Download the gwa_calculator.py file
 2. Run the program with Python 3:
 3. Calculate and save GWA for a quarter
 4. View saved GWA records
 5. Exit the program
